# agul-cli
![Static Badge](https://img.shields.io/badge/agul-cli_v1.0.0-green)

## 📖 简介
- 快速搭建前端项目模板的脚手架工具

## 📝 模板
- vue3技术栈模板 
Vue3+TS+Vite+ElementPlus+Pinia+Axios+Uniapp
- ✅PC  
  - geeker-admin、plus-ui
- ✅Mobile 
  - unibest

## 📦 安装
```bash
npm install agul-cli -g
```

## 🔨 使用
```bash
# 创建模板项目
agul create <project-name>

# 选择模板

# 查看脚手架版本
agul -v
agul --version

# 更新脚手架到最新版本
agul update
```
